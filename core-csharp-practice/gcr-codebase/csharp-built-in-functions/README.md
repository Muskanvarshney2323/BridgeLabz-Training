# csharp-built-in-functions Exercises

This folder contains small C# console programs demonstrating built-in DateTime operations and other algorithmic exercises.

Files:

- `TimeZonesDateTimeOffset.cs` - Display current times in GMT, IST, PST using DateTimeOffset and TimeZoneInfo (with fallbacks if system IDs are unavailable).
- `DateArithmetic.cs` - Show adding 7 days, 1 month, 2 years to a given date and then subtract 3 weeks (using AddDays(-21)).
- `DateFormatting.cs` - Display the current date in `dd/MM/yyyy`, `yyyy-MM-dd`, and `EEE, MMM dd, yyyy` formats.
- `DateComparison.cs` - Take two dates and report whether the first is before, after, or same as the second.
- `NumberGuessingGame.cs` - Computer tries to guess a user-thought number using random guesses and feedback functions.
- `MaxOfThreeNumbers.cs` - Find the maximum of three integers using modular functions.
- `PrimeNumberChecker.cs` - Check if a number is prime (uses separate IsPrime function).
- `FibonacciSequenceGenerator.cs` - Generate Fibonacci sequence up to a specified number of terms.
- `PalindromeCheckerBuiltin.cs` - Check whether a string/phrase is a palindrome (ignores non-alphanumeric characters and case).
- `FactorialRecursion.cs` - Compute factorial using a recursive function (uses BigInteger).
- `GcdLcmCalculator.cs` - Compute GCD and LCM of two integers (separate functions for each).
- `TemperatureConverter.cs` - Convert temperatures between Celsius and Fahrenheit.
- `BasicCalculator.cs` - Perform basic arithmetic operations (+ - * /) with separate functions for each operation.

Each file contains a `Main` method with console I/O and demonstrates parameter passing/returns where applicable.
